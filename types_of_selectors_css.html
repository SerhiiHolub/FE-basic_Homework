Універсальні селектори:
* {
font-size: 16px;
}
 Універсальні селектори CSS позначаються звичайною зірочкою "*". Під зірочкою може матися на увазі будь-який елемент розмітки HTML-сторінки.

Селектори за назвою тега:
h1 {
color: red;
}
 Запис селекторів за тегами де h1 – селектор за назвою тега.

Селектори за класом:
.main-header {
background-color: #99ffcc;
}
 "class" - це атрибут HTML-тега, він дуже зручний тим, що може застосовуватись до однотипних елементів. Назва класу може бути будь-яким, головне, щоб вона відповідала тому елементу, який описується CSS-правилом. Взагалі атрибут "class" вважається найголовнішим інструментом CSS. При створенні CSS-правила точка "." перед ім'ям класу є обов'язковим.

Селектори з id:
#heading {
font-size: 35px;
}
 До атрибутів HTML-тегів і id. Цей атрибут застосовується до індивідуального елементу, тобто. елемент з id може бути в єдиному екземплярі на сторінці згідно з правилами семантики. У каскадних таблицях стилів ці атрибути використовуються вкрай рідко, їх застосування до них взагалі не вітається, вони призначені в основному для роботи з javascript.

Селектори за атрибутом:
a[href = "https://loftblog.com/"] {
color: black;
}
 CSS селектори за атрибутами HTML-тегів.

Селектори нащадків, або контекстні селектори:
.wraping p {
padding: 15px;
}
 Під CSS селекторами нащадків, або контекстними селекторами, мається на увазі ситуація, коли один елемент HTML-сторінки вкладений в інший, причому не обов'язково він повинен бути першим нащадком.

Селектори дочірні (тільки перші):
.wraping>p {
`padding: 15px;
}
Дочірні селектори CSS відрізняються від селекторів нащадків тим, що під дочірнім мається на увазі лише перший нащадок, ніякі "онуки" та "правнуки" не потрапляють до цієї категорії.

Селектори сестринські, або сусідні (розташовані на одному рівні):
h1 + p {
padding-bottom: 30px;
}
 За допомогою сестринського селектора створюється CSS-правило для HTML-елементів, розташованих на одному рівні, причому той елемент, який записується як сестринський, повинен бути найближчим із себе подібних до того елемента, до якого він записується в "сестри".

Селектори псевдокласів (селектори стану):
a:hover {
text-decoration: none;
}
 До селекторів псевдокласів відносяться селектори CSS, які описують стан елементів при здійсненні над ними будь-якої дії. Це може бути наведення елемент курсору миші, клік по елементу тощо. Записуються вони через двокрапку до елемента, до якого застосовуються.

Селектори псевдоелементів:
p::first-letter{
color: red;
}
 За допомогою (::before, ::after) можна додавати різні елементи на веб-сторінки, але при цьому псевдоелементи ніяк не будуть відображені в HTML-розмітці цих сторінок. Записуються вони по відношенню до елемента через подвійну двокрапку.
