Команди для роботи з терміналом

  • git status-дивитись зміни
  • git add . (додати все)
  • git commit -m "add index file"-автоматичне зберігання тексту
  • git push origin-записати зміни у гітхаб
  • git remote -v-версія
  • git branch-дивитись яка гілка
  • git checkout -b-перейти на якусь гілку
  • git checkout-змінити гілку
  • git push origin HEAD-Створити голову гілки
  • git branch -D-Назва гілки (Видалення гілки)
